# Poker Chip Tracker

A simple web app for tracking poker chips when playing with friends without physical chips.

## Features

- Add/remove players
- Track chip balances
- Bet chips into a shared pot
- Collect winnings (full pot or split)
- Fold players during a hand
- Quick bet buttons ($10, $25, $50, $100, All-In)
- Poker hand rankings reference
- Mobile-friendly design

## Quick Start

```bash
pip install -r requirements.txt
python main.py
```

Open http://localhost:5001 in your browser.

## Usage

1. Add players with the input form
2. Click **Bet** to move chips to the pot
3. Click **Win** to collect from the pot
4. Click **Fold** to mark a player as out for the hand
5. Click **New Hand** to reset folds for the next round
6. Click **See Rules** to view poker hand rankings
